## ⚔️ 這是什麼

《天下布武》是一款**純瀏覽器、單機、全繁體中文**的戰國大戰略遊戲。
你扮演一位戰國大名，在整個日本的版圖上分封知行、開發領國、調略人心、締結姻盟、出陣合戰——
最終一統天下。玩法向《信長之野望·新生》致敬，機制則全部重新設計。

## ✨ 核心特色

- 🏯 **知行與家臣自主** — 你不必微管理每一塊土地。把郡分封給武將，他們便自主治理、開發、成長；你只需掌握大方向。
- 🗾 **即時天下大勢** — 全日本地圖上，各勢力同時即時行動（可暫停、變速 ×1／×2／×5）。戰線推移與勢力消長，一目瞭然。
- 💥 **威風的爽快感** — 打贏一場關鍵合戰，鄰近敵郡望風而降、敵城士氣崩落，戰線一口氣大幅推進。
- 🎎 **武將是資產** — 登用、育成、任命、忠誠、引拔與叛服……六階身分與功績累積，是長期經營的核心。
- 📜 **歷史沉浸** — 桶狹間、川中島、本能寺（條件式）等史實事件；朝廷官位、室町幕府，與戰國語感的繁中文本。

**系統一覽**：內政（郡開發／知行／城下施設／政策／徵兵／輸送／治安一揆）· 軍事（編成出陣／行軍／制壓／野戰自動解算／合戰戰術戰場／戰法／威風／攻城／軍團／兵站）· 外交（外交工作與信用／同盟／婚姻／停戰／從屬）· 朝廷與幕府（官位八階／役職五階）· 調略（引拔／流言／內應）· 具申（家臣提案）· 歷史事件與大命 · 勝敗判定與結局 · 存讀檔。

## 🤖 願景：讓 AI 執掌天下

**這才是本作的核心野望。** 從第一行架構就為 AI 而設計。

遊戲核心是一個**純粹、確定性的狀態機**：

```
(遊戲狀態 GameState, 指令 Command[]) ──每日 tick──▶ 新狀態 + 事件 GameEvent[]
```

不論是人類玩家還是 AI，操作一位大名的**唯一方式都是送出指令（Command）**。於是：

- 🧠 **每一位大名，都能交給 AI 操盤。** 你可以只執掌其中一家，其餘四十家由 AI 各自經營、彼此攻伐；也可以完全旁觀，看一整盤天下大勢自行演化。
- 🎭 **每家 AI 都有性格（AiPersona）**：征服者、守成者、謀略家……不同性格帶來不同的擴張、外交與用人風格；委任的城主、領主、軍團也各有其代管 AI。
- 🔬 **內建無頭模擬器（`tools/simulate.ts`）**：不開畫面、全 AI 跑上數十年，輸出各家石高曲線——既用來驗證平衡，也讓你回看「如果沒有你，歷史會怎麼走」。
- 🚀 **未來展望**：因為決策層（`AiIntent`）與規則層乾淨分離，大名的「大腦」可以整顆抽換成更強的 AI——**甚至接入大型語言模型（LLM）**，讓每位大名成為會權衡、會結盟、會背叛、有獨立性格的智能體。屆時，這張戰國地圖就是一座 AI 的競技場。

## 🛠️ 技術架構

- **前端**：TypeScript（strict）＋ React 18 ＋ Vite；狀態管理 Zustand；地圖以 **PixiJS 8（WebGL）** 渲染上千節點、穩定 60fps。
- **純前端 SPA、零後端**，可直接部署到 GitHub Pages。存檔走 IndexedDB（idb-keyval）＋ lz-string 壓縮。
- **核心鐵律**：`src/core/` 是零 DOM 依賴的純邏輯，能在 Node 裡跑測試與模擬；**決定論設計**（固定亂數流），同劇本同種子必重現同一結果——這也是 AI 對戰可重播、可驗證的基礎。
- 全 UI **繁體中文（台灣慣用語）**，字形嚴禁簡體與日文新字體。
