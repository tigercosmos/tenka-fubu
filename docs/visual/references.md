# 《天下布武》視覺參考圖板與禁抄紀錄

> - 建立日期：2026-07-14
> - 用途：支援 M6-V1 Art Bible 與後續 visual fixture 研究
> - 原則：只保存網址與文字觀察，不在 repository 內保存、裁切、描摹或再散布參考畫面。

## 1. 使用方式

本表以《信長之野望・新生／NOBUNAGA'S AMBITION: Awakening》的官方網站、官方手冊與發行商新聞頁為主要觀察入口。參考價值只在於理解大地圖戰略遊戲的資訊問題，不授權沿用其視覺資產。

每一筆研究依下列流程處理：

1. 確認來源網址、來源類型與存取日期。
2. 以文字記錄「玩家需要讀懂什麼」及「哪些資訊在某縮放仍存在」。
3. 將觀察轉譯為 `art-bible.md` 的原創紙雕、墨線、旗幟、圖示與 HUD 規則。
4. 明列不可直接沿用的構圖、輪廓與資產，避免把 mood board 變成臨摹清單。
5. 後續草稿只從本作規格出發；參考畫面不進 atlas、測試 fixture 或 production bundle。

## 4. 禁用與來源管理

### 4.1 repository 禁入項

- 上述網站的圖片下載檔、截圖、影片 frame、裁圖、縮圖與色票擷取結果。
- 以原作畫面為底的描摹稿、vector trace、輪廓翻轉／改色版本。
- 原作角色、家紋、城池、兵種、地圖、圖示或 UI 的生成式近似品。
- 沒有作者、來源、授權、衍生狀態與內容 hash 的外部素材。

### 4.2 可保存項

- 本文件中的 URL、存取日期與抽象文字觀察。
- 從 `art-bible.md` 出發的原創灰階 wireframe、幾何 construction sheet 與 icon grid。
- 自製程序紋理、原創 SVG／bitmap，以及完整登錄於素材 manifest 的合法資產。
- 固定 seed 的本作 visual fixture 與測試截圖；baseline 只能包含本作自有畫面。

### 4.3 新增參考的必要欄位

新增任何來源時，必須填寫：

- 唯一 ID 與場景／問題。
- 可直接開啟的來源 URL、發布者／來源類型、存取日期。
- 不包含像素尺寸、色碼與可臨摹描述的抽象觀察。
- 對應的本作原創轉譯與預定測試。
- 明確的「不可直接沿用」內容。

若來源頁失效，保留 URL 與文字紀錄並標記失效日期；不得以來路不明的鏡像站或圖片搜尋縮圖取代。

### 4.4 來源可及性紀錄

- **2026-07-22（M6-V9b 研究）**：嘗試存取 `qbnews.tw/shinsei`（台灣遊戲媒體之新生介面圖文）作為
  介面觀察補充，遭 Cloudflare 人機驗證阻擋（無法程式化存取）；依 §4.3 流程改以官方 KOEI TECMO
